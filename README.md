# Juego Pong

Juego clásico de Pong hecho con Python y turtle.

## ¿Qué hace?

Reproduce el juego Pong — dos jugadores, una pelota y un
marcador de puntos.

## Tecnologías

- `turtle` — dibuja y anima los elementos del juego
  (incluido en Python)

## Uso

```bash
python main.py
```

## Controles

| Jugador   | Arriba | Abajo |
| --------- | ------ | ----- |
| Izquierda | W      | S     |
| Derecha   | ↑      | ↓     |

## ¿Cómo funciona?

- La pelota rebota en las paredes de arriba y abajo
- Si la pelota pasa una paleta, el otro jugador suma un punto
- La pelota vuelve al centro después de cada punto
